Wallpaper

To set a custom background image on the new tab page, navigate to the appropriate direcotry (listed below; make sure to change USERNAME to your user name) and find the extension's folder named Replace the "wallpaper" file with any image if your choice. Please make suer to remove the extension of the image by renaming it. It should be named just "wallpaper", not "wallpaper.jpg"

Windows XP 
C:\Documents and Settings\USERNAME\Local Settings\Application Data\Google\Chrome\User Data\Default\Extensions
Windows Vista 
C:\Users\USERNAME\AppData\Local\Google\Chrome\User Data\Default\Extensions
Windows 7 
C:\Users\USERNAME\AppData\Local\Google\Chrome\User Data\Default\Extensions
Mac OS X 
/Users/username/Library/Application Support/Google/Chrome/Default/Extensions

Features

  • Completely rewritten The original extension was built off of New tab w/ clock and current project entry by kalebnyq but it evolved into something much more than a simple clock. A rewrite has been long overdue to make it more efficient and easier to use.
  • More customizability Many more options to make it work the way you want. Translations feature allows you to customize all the text on the New Tab page. Pair this with custom CSS themes and you can turn it into anything you want.
  • New look The New Tab page has a much cleaner look and the Options page is completely redesigned.
